# FlowCross Launcher

**Дев:** lowfreez

FlowCross — лаунчер для запуска и управления версиями Minecraft.  
Проект ориентирован на простой интерфейс для выбора профиля, версии игры и параметров запуска, а также на базовое управление локальными файлами игры.

Лаунчер не является официальным клиентом Minecraft и не распространяет игровые файлы.

---

## Возможности

- Управление профилями запуска (имя игрока, выделенная память и параметры JVM).
- Просмотр списка доступных версий игры из официального публичного манифеста Mojang.
- Установка и запуск версий по запросу пользователя.
- Настройка путей (каталог лаунчера, папка Minecraft).
- Автоматическое создание стандартных каталогов (`mods`, `resourcepacks`, `versions` и др.).
- Локальное хранение настроек и профилей.
- Проверка обновлений через `electron-updater`.
- Экран загрузки (splash) и индикатор прогресса в панели задач (Windows).

---

## Архитектура

- **Electron main process**  
  Управляющая логика, работа с файловой системой, сетевые запросы, создание окон и IPC.

- **Renderer (HTML / CSS / JavaScript)**  
  Пользовательский интерфейс и навигация.

- **Preload**  
  Безопасный мост между UI и основным процессом.

- **Хранилище**  
  `electron-store` для сохранения настроек и профилей.

---

## Сетевые ресурсы

- Список версий загружается из публичного манифеста Mojang:  
  https://launchermeta.mojang.com/mc/game/version_manifest.json
- Дополнительные ресурсы (иконки, splash) могут загружаться извне и сохраняться локально.
- Для загрузок используется стандартный модуль `https`.

---

## Интерфейс

- Интерфейс реализован на HTML/CSS.
- Основные элементы используют SVG-иконки.
- Поддерживается смена состояния иконок (например, активное / неактивное).

---

## Ограничения

- Лаунчер не заменяет официальные сервисы Minecraft.
- Автоматическая установка модов не реализована.
- Проверка установленных версий выполнена в упрощённом виде.
- Работа обновлений зависит от конфигурации `electron-updater`.

---

## Платформы и требования

- Поддерживаемые платформы: **Windows, macOS, Linux**.
- Для запуска игры требуется установленная Java (обнаруживается автоматически или указывается вручную).

---

## Диагностика

- Логи сохраняются локально в каталоге `userData/logs`.
- При сбоях рендер-процесса окно может быть перезапущено.

---

## Разработка

- Проект использует JavaScript, HTML, CSS и Electron.
